# CSV Widget for ArcGIS Flex Viewer

Uploads a CSV file to the viewer and plots those coordinates in the file onto the viewer. The coordinates
can be in NZTM, NZMG or WGS coordinate systems.


## Features

* Upload CSV file eith coordinates
* NZTM, NZMG and WGS coordinate systems supported
* Show points on map


## Requirements

* Flash Builder 4
* Flex SDK 4.6


## Installation Instructions

* Fork and then clone the repo or download the .zip file. 
* Add this into the main config file for the ArcGIS Flex Viewer:

`<widget label="Load CSV"
                icon="assets/images/CSV.png"
                config="widgets/CSV/CSVWidget.xml"
                url="widgets/CSV/CSVWidget.swf"/>`
